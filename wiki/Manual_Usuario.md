# 📖 Manual de Usuario

Guía para dominar el sistema de comunicación bidireccional de **El Séquito Edition**.

## 1. Configuración Inicial
Accede al panel de control escribiendo `/pfui` y navegando a la sección **Translator** (Traductor).

### Opciones de Dirección (Direction)
Esta es la característica nuclear de la versión v1.0.0:
- **Auto-Detección**: El motor decide si traducir de ES a EN o viceversa.
- **Español -> Inglés**: Forzar traducción de tus mensajes al inglés (Ideal para grupos internacionales).
- **Inglés -> Español**: Forzar traducción de tus mensajes al español (Para canales específicos).

## 2. Comandos de Chat
Puedes usar el comando raíz `/tr` para las siguientes acciones:
- `/tr`: Abre directamente la configuración del traductor.
- `/tr stats`: Muestra en el chat las estadísticas de rendimiento y mensajes procesados.

## 3. Integración con Canales
Puedes habilitar o deshabilitar la traducción para canales específicos (Say, Guild, World, LFG). Se recomienda mantener **World** y **LFG** activos para una experiencia óptima en Turtle WoW.
