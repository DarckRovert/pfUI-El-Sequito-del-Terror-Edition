# API Guide — pfUI Sequito 🛠️

Este fork expone hooks adicionales para el WCS_Brain.

### `pfUI.api.GetSequitoSkin()`
Retorna los parámetros de color y bordes actuales del tema "El Séquito".

### `pfUI.api.ApplySequitoStyle(frame)`
Aplica el borde y fondo personalizado del clan a cualquier frame externo.
