# Contributing to El Séquito del Terror Ecosystem

Thank you for your interest in contributing! This project is a community-driven open-source addon suite for Turtle WoW, developed by **DarckRovert (Elnazzareno)** and the **El Séquito del Terror** clan.

## How to Contribute

### Reporting Bugs
- Open an issue on GitHub with a clear description of the bug.
- Include your WoW client version (Turtle WoW 1.12.1) and steps to reproduce.

### Suggesting Features
- Open an issue with the label enhancement.
- Describe the feature and how it fits the ecosystem's philosophy.

### Submitting Code
1. Fork the repository.
2. Create a new branch: git checkout -b feature/my-feature.
3. Make your changes following Lua 5.0 compatibility standards.
4. Commit with a clear message: git commit -m "feat: add new feature".
5. Push to your fork and open a Pull Request.

## Code Standards
- **Lua 5.0**: All code must be compatible with the Turtle WoW client (WoW 1.12.1 engine).
- **No globals pollution**: Use the addon's namespace (e.g., WCS_Brain) for all variables.
- **Comments in Spanish or English** are both welcome.
