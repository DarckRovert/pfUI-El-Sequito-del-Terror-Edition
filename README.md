## 🎨 ¿Qué hace especial a esta edición?

Esta no es la versión estándar de Shagu's pfUI. La **El Séquito del Terror Edition** incluye:

- **Configuración Integrada**: Perfiles pre-cargados que optimizan el espacio para el HUD de `WCS_Brain`.
- **Skins Exclusivos**: Texturas y bordes personalizados con la estética del clan.
- **Micro-Optimización**: Scripts de limpieza de memoria específicos para el cliente de Turtle WoW.
- **Warlock Ready**: Configuración de marcos de unidad optimizada para el seguimiento de mascotas y Soul Shards.

## 🚀 Instalación & Uso

1.  Extrae en `Interface\AddOns\pfUI\`.
2.  Al entrar al juego, el asistente de pfUI detectará automáticamente el ecosistema **WCS**.
3.  Carga el perfil de **"El Séquito"** desde `pfUI -> Perfiles`.
