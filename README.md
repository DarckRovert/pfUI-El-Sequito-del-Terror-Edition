## 🖥️ ¿Qué es pfUI?

pfUI reemplaza completamente la interfaz predeterminada de WoW 1.12.1 con un diseño minimalista y moderno. Barras de acción rediseñadas, marcos de unidad limpios y una estética compatible con el universo oscuro del Séquito del Terror.

**Crédito original:** Shagu — [github.com/shagu/pfUI](https://github.com/shagu/pfUI)

## 🚀 Características

- Marcos de unidad rediseñados (jugador, objetivo, grupo, raid).
- Barras de acción personalizables (posición, tamaño, transparencia).
- Mapa del mundo y minimapa mejorados.
- Fuente y colores del Séquito.
- Integración con pfQuest para marcadores de misión.
- Compatible con TerrorMeter y WCS_Brain ButtonBar.

## ⚙️ Instalación

1. Extrae en Interface\AddOns\pfUI\.
2. Activa el addon. **IMPORTANTE:** desactiva los addons de UI que conflictúen (CT_MOD, Discord UI, etc.).
3. Escribe /pfui para abrir la configuración.
