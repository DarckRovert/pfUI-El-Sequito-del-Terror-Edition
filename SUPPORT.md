# Soporte Técnico

Si necesitas ayuda con el Traductor Global, sigue estos canales oficiales:

## 1. Documentación (Primer Paso)
Revisa la [Wiki Técnica](file:///e:/Turtle%20Wow/Interface/AddOns/pfUI/wiki/FAQ.md). La mayoría de los problemas de configuración se resuelven aquí.

## 2. Twitch Stream (Comunidad)
Visita el canal de [DarckRovert](https://twitch.tv/darckrovert) durante los directos para resolver dudas en tiempo real con la comunidad del Séquito.

## 3. GitHub Issues (Errores Técnicos)
Si has encontrado un fallo reproducible, abre un [Issue](https://github.com/DarckRovert/Global-Chat-Translator/issues) siguiendo la plantilla correspondiente.

---
No se proporciona soporte para versiones modificadas o "leaked" fuera del ecosistema oficial.
